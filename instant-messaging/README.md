# 即時通訊

* WebSocket：即時的 socket 連線
* Web Workers：以往 JavaScript 都是 single thread，透過 Worker 可以有多個運算
* Notifications：原生的提示訊息，類似像 OS X 的 Growl 提示

```
Notification.requestPermission()
```

```
Notification.permission
```

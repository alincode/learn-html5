overflow

用來定義元素超過某個範圍的時候該如何呈現，例如圖片超過預設區域的大小、文字長度超出原本的範圍等

CSS overflow 常見的值

  • overflow: auto;　// 預設會自動使用捲軸
  • overflow:visible;　 // 顯示的文字或圖片會直接超出範圍，不使用捲軸。
  • overflow:hidden;　// 自動隱藏超出的文字或圖片。
  • overflow:scroll;　// 自動產生捲軸。
  • overflow:inherit;　// 繼承自父元素的可見性

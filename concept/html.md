# HTML5

* 不分大小寫

**HTML5改善了哪些**

* 將 vidoe 和 audio 分開定義
* 加入了多種 input type，例如 url, email
* 廢棄與 CSS 功能重複性高的元素，例如 u, font, center
* 重新定義 b, i 元素含義

**HTML5宣告**

* `<DOCTYPE>` 不需要結束標籤

```
<!DOCTYPE HTML>
<html>
<head>
<title>Title of the document</title>
</head>
<body>
The content of the document......
</body>
</html>
```

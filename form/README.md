# 表單設計

**觸發驗證的時機點**

## HTML5新增的input類型##

* color
* date
* datetime
* datetime-local
* email
* month
* number
* range
* search
* tel (The tel type is currently supported only in Safari 8.)
* time
* url(not supported in IE9 and earlier versions.)
* week

* file

```
<input type="file" name="upfile[]"/>
```
